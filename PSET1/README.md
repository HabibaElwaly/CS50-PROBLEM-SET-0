# CS50-PROBLEM-SET-1
My solutions for CS50x Problem Set 1,implemented in C language:
* **hello.c** : A simple C program that asks for the user's name and prints a personalized greeting.
* **mario.c** : A program that builds Mario's double pyramids using hashes (#).
* **credit.c** : A C program that validates credit card numbers using Luhn's Algorithm and determines the card issuer (AMEX, MASTERCARD, or VISA).
